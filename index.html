About

I'm a Full Stack JavaScript Web Developer located in Tucson, AZ. I'm a native of Arizona and graduated from Arizona State University with a degree in Geography. I went on to work with the U.S. Geological Survey for 21 years as an Information Technology Specialist with a focus on Geographic Information Systems (GIS). I was also a team member in the development of the National Geologic Map Database. I believe the combination of GIS technology and web development technologies can combine to create very interesting applications.

Some of the technologies that I enjoy using are:

Front-end: AJAX Bootstrap HTML5 CSS3

Back-end: Javascript Node.js

Others NPM (Node Package Manager) GIT (Version Control) Github Third party APIs JSON
